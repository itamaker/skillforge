# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| `0.1.x` | yes |

## Reporting a Vulnerability

If you find a security issue, please do not open a public issue first.

Report it privately through GitHub Security Advisories or contact the maintainer directly, and include:

- affected version
- impact summary
- reproduction details
- any suggested mitigation

You can expect an initial response within a reasonable timeframe after the report is received.
